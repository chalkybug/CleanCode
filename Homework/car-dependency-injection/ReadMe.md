2. hệ thống điều khiển tự lái thông minh => thay đổi trong lúc chạy

        2. manual steering
3. có thể có tuỳ chọn động cơ xăng hoặc điện tại thời điểm lắp ráp (constructor dependency injection).
    - Engine  có 2 loại:
      1. gas engine
      2. electric engine
